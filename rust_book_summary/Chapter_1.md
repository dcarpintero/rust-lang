# Installation

Basic Rust Commands:

```
    rustc --version
    rustup update
    rustup doc             // local documentation
```

Basic Cargo Commands:

```
    cargo new hello_cargo  // create project
    cargo check            // make sure it compiles (much faster)
    cargo build            // compile, create executable in target/debug
    cargo run              // compile and run
    cargo build --release  // build for optimizations, create executable in target/release

    cargo doc --open       // project documentation
```
